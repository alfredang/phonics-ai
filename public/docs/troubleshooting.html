Troubleshooting Guide

Solutions to common problems

Quick Fixes

Before diving into specific issues, try these general solutions that fix most problems:

  1. Refresh the page

    Press F5 or click the refresh button in your browser

  2. Clear browser cache

    Press Ctrl+Shift+Delete (Windows) or Cmd+Shift+Delete (Mac)

  3. Try a different browser

    Chrome is recommended for the best experience

  4. Check your internet connection

    Make sure you're connected to WiFi or mobile data

Audio Issues

🔈 No Sound Playing

Symptoms: Clicking speaker buttons doesn't produce any sound

Solutions:

  1. Check device volume - Make sure your device isn't muted and volume is turned up
  2. Check browser tab - Look for a muted speaker icon on the browser tab. Click it to unmute
  3. Check headphones - If using headphones, ensure they're properly connected
  4. Test other websites - Play audio on YouTube to verify your speakers work
  5. Restart browser - Close all browser windows and reopen
Tip: If audio works in other apps but not in Phonics AI, try a different browser. Chrome provides the best audio support.

🔈 Sound Quality Issues

Symptoms: Audio is distorted, choppy, or unclear

Solutions:

  1. Use headphones - Built-in speakers may not reproduce sounds clearly
  2. Check internet speed - Slow connections can cause audio stuttering
  3. Close other tabs - Too many open tabs can affect performance
  4. Update browser - Older browsers may have audio issues

Microphone Issues

🎤 Microphone Not Working

Symptoms: Voice recording features don't respond or show errors

Solutions:

  1. Grant microphone permission

    When prompted, click "Allow" to let the app use your microphone

  2. Check browser settings
    • Chrome: Settings → Privacy and security → Site settings → Microphone
    • Safari: Preferences → Websites → Microphone
    • Firefox: Settings → Privacy & Security → Permissions → Microphone
  3. Close other apps using microphone

    Only one app can use the microphone at a time

  4. Check microphone hardware

    Test your microphone in another app (like Voice Memos) to ensure it works

  5. Use Chrome browser

    Chrome has the best support for Web Speech API features

Note: Voice recording features require a working microphone and may not be available on all devices or browsers.

Login & Account Issues

🔒 Can't Log In

Symptoms: Unable to access your account

Solutions:

  1. Check email address - Ensure you're using the correct email
  2. Check password - Make sure Caps Lock isn't on
  3. Reset password - Click "Forgot Password" to reset
  4. Try Google sign-in - If you registered with Google, use the Google button
  5. Clear cookies - Old login data might be causing conflicts

🔒 Forgot Password

Symptoms: Can't remember your password

Steps to Reset:

  1. Go to the login page
  2. Click "Forgot Password"
  3. Enter your email address
  4. Check your email (including spam folder)
  5. Click the reset link in the email
  6. Create a new password
Tip: If you don't receive the reset email within a few minutes, check your spam folder or try again.

Progress & Data Issues

📊 Progress Not Saving

Symptoms: Completed lessons don't show as completed after refresh

Solutions:

  1. Verify you're logged in - Progress only saves when logged into an account
  2. Check internet connection - Progress syncs to the cloud when online
  3. Wait a moment before closing - Allow time for data to sync after completing a lesson
  4. Don't use private/incognito mode - Progress may not persist in private browsing

📊 Progress Disappeared

Symptoms: Previous progress no longer shows

Solutions:

  1. Check you're on the right account - You might have multiple accounts
  2. Verify the email address - Ensure you logged in with the same email
  3. Log out and log back in - This can refresh your data from the server
  4. Wait and refresh - Sometimes data takes time to sync
Important: If you used the app without logging in, that progress cannot be recovered. Always create an account to save your progress!

Display & Performance Issues

💻 App Loading Slowly

Symptoms: Pages take a long time to load or feel sluggish

Solutions:

  1. Check internet speed - Run a speed test at speedtest.net
  2. Close unused browser tabs - Too many tabs consume memory
  3. Restart your browser - Clears memory and cached data
  4. Update your browser - Older versions may be slower
  5. Disable browser extensions - Some extensions can slow down websites
  6. Try a different device - Older devices may struggle with animations

💻 Display Not Rendering Correctly

Symptoms: Elements look broken, overlapping, or missing

Solutions:

  1. Zoom to 100% - Press Ctrl+0 (Windows) or Cmd+0 (Mac)
  2. Clear browser cache - Old CSS files might be cached
  3. Try a different browser - Chrome is recommended
  4. Disable dark mode extensions - They can interfere with app styling
  5. Check screen resolution - Very low resolutions may cause issues

Mobile-Specific Issues

📱 Touch Not Responding

Symptoms: Tapping buttons or dragging elements doesn't work

Solutions:

  1. Refresh the page - Pull down to refresh on mobile
  2. Close and reopen browser - Fully close the app and restart
  3. Clear browser data - Go to browser settings and clear cache
  4. Restart your device - Sometimes a reboot is needed

📱 Screen Too Small

Symptoms: Content is hard to see or tap on smaller screens

Solutions:

  1. Use landscape mode - Rotate your device sideways
  2. Use a tablet instead - Larger screens provide a better experience
  3. Use a computer - Desktop provides the best experience for games
